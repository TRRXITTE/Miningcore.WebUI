Mining Tutorial

Step-by-step setup guide

Getting Started with Monero Mining

  1. Select a mining pool (e.g., multipool.dev)
  2. Register and get your XMR wallet address
  3. Choose hardware (CPU recommended, GPU optional)
  4. Download RandomX-compatible mining software (see below)
  5. Configure with pool details and difficulty settings
  6. Monitor performance and earnings

Tips: Use high-core-count CPUs (e.g., Ryzen 9) for best results; adjust difficulty based on your hardware.

Miner Configurations

Examples for RandomX miners with difficulty options

XMRig (CPU/GPU) - Low Difficulty (1000+)
xmrig -o stratum+tcp://bitcoin.multipool.dev:3032
    -u YourXMRWalletAddress.WorkerName 
    -p x 
    --randomx-mode=auto 
    --cpu-max-threads-hint=75
XMRig (CPU/GPU) - Medium Difficulty (10000+)
xmrig -o stratum+tcp://bitcoin.multipool.dev:3033
    -u YourXMRWalletAddress.WorkerName 
    -p x 
    --randomx-mode=fast 
    --cpu-max-threads-hint=85
XMRig (CPU/GPU) - High Difficulty (200000+)
xmrig -o stratum+tcp://bitcoin.multipool.dev:3034
    -u YourXMRWalletAddress.WorkerName 
    -p x 
    --randomx-mode=light 
    --cpu-max-threads-hint=100
SRBMiner-Multi (CPU/GPU)
srbminer-multi --algorithm randomx 
    --pool stratum+tcp://bitcoin.multipool.dev:3033
    --wallet YourXMRWalletAddress.WorkerName 
    --password x 
    --cpu-threads 8
