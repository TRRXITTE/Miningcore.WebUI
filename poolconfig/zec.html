Mining Tutorial

Step-by-step setup guide

Getting Started with Zcash Mining

  1. Select a mining pool (e.g., multipool.dev)
  2. Register and get your ZEC wallet address
  3. Choose Equihash-compatible hardware (ASICs like Antminer Z15 or GPUs)
  4. Download mining software (see below)
  5. Configure with pool details and difficulty settings
  6. Monitor performance and earnings

Tips: ASICs are most efficient; GPUs work for smaller setups. Ensure good cooling and stable power.

Miner Configurations

Examples for Equihash miners with difficulty options

EWBF Miner (GPU) - Variable Difficulty (0.01+)
miner --server bitcoin.multipool.dev 
    --port 3092 
    --user YourZECWalletAddress.WorkerName 
    --pass x 
    --algo equihash
GMiner (GPU) - Variable Difficulty (0.01+)
gminer --algo equihash 
    --server bitcoin.multipool.dev 
    --port 3092 
    --user YourZECWalletAddress.WorkerName 
    --pass x
BMiner (ASIC/GPU) - Static Difficulty (1)
bminer -uri stratum+tcp://YourZECWalletAddress.WorkerName:x@bitcoin.multipool.dev:3093 
    -api 127.0.0.1:1880
lolMiner (GPU) - Variable Difficulty (0.01+)
lolMiner --algo EQUIHASH 
    --pool stratum+tcp://bitcoin.multipool.dev:3092 
    --user YourZECWalletAddress.WorkerName 
    --pass x
